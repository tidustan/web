GIMP Portable Help

Image Editing Made Portable

GIMP Portable allows you to carry a complete image editor with you on an iPod, USB flash drive, portable hard drive, CD or any other portable media. You can plug it right into any Windows computer and use it just like you would on your own. It is a repackaged version of the popular the GIMP for Windows image editor designed with portability in mind, so it has all the same great features of the GIMP, but there's nothing to install.
